**Création d'un site web dynamique avec Javascript**

Ce projet avait pour but de finaliser le site Internet d'une architecte en utilisant Javascript et une API HTTP.

Principales fonctionnalités développées :

-> Récupération de données via une API HTTP et des formulaires

-> Connexion et déconnexion de l'utilisateur

-> Création et gestion de modales pour la gallerie photos

-> Validation des saisies utilisateur et gestion des erreurs (structures try/catch)

-> Gestion des événements utilisateurs en utilisant la méthode AddEventListner et, le cas échéant, la délégation d'événements

-> Manipulation d'éléments du DOM

-> Application des bases de l'algorithmique (variables, fonctions, boucles, conditions, récursivité)
